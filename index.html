Default parameters

In JavaScript, the default value for function parameters is undefined.

It may be handy to set a different value instead. This can be achieved with the help of default function parameters.

Before ES6,
function add(num1, num2) {
return num1+num2;
}
add (5,2); //returns 7
add(3); //returns NaN as num2 is undefined

With ES6,
function add(num1, num2=10) {
return num1+num2;
}
add (5,2); //returns 7
add(3) //returns 13 as num2 has default value = 10

With ES6, earlier function parameters are available for the later default parameters as well
function getFullName(firstName, lastName, fullName=lastName+' '+lastName){
return fullName;
}
getFullName('John','Doe'); // returns John Doe
getFullname('John', 'Doe', 'John Doe'); // returns John Doe
